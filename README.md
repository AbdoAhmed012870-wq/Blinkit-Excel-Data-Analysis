# Blinkit Sales & Retail Data Analysis (Excel)

## Project Overview
A comprehensive data analysis and business intelligence solution built entirely in Microsoft Excel to analyze Blinkit's retail sales, product performance, and outlet efficiency.

## Key Insights & Technical Features
* **Advanced Formulas:** Utilized complex functions including `VLOOKUP`, `INDEX-MATCH`, and nested `IF` statements for seamless data transformation.
* **Data Modeling:** Built dynamic `Pivot Tables` and `Pivot Charts` to aggregate and summarize over thousands of sales transactions.
* **Interactive Dashboard:** Designed a professional, user-friendly Excel dashboard with interactive slicers to filter data by region, outlet type, and time periods.
